
# - 可以解决红线问题
-

随着数据量增长，如何处理价值不太大的老旧数据？

最近出现新的解决方案，用关系数据库存储最新数据，同时使用Hadoop存储老数据
数据要么以文件形式存储在HDFS上，要么存储在Hive并保留RDBMS接口。
